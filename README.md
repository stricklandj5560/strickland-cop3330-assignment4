# User Guide

- Getting Started: Upon launching the application, you will be presented with a "My To Do List" window. First, select "Add New" to add a new todo list.

- Once you've added a new todo list, you can edit that list individually, first give it a name by selecting the title and editing the name. Then, select the "Add New Item" button to start adding items to you todo list!

- Once you've added a new item to your todo list, give it a name and a due date, and once it's completed, you can mark the item as complete by selecting the "Completed" checkbox.

- At any time, you can select any cell and either select the "Remove Selected Cell" button to remove the todo item, or you can edit that item's due date or description.

- If you wish to delete a todo list, simply select the "Delete List" button.

- You also have the option to select "Save List" to save that list to an external storage.

- To load a todo list, simply select the "Load" button, and load in the todo list file from external stoarge.
